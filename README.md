# Tucil1_18223057

## Penjelasan Program secara Singkat

Program ini merupakan realisasi perancangan solusi dari N Queen LinkedIn Problem (N Queen Problem + Constraint Warna) dengan pendekatan berjenis *brute force* di bahasa python (.py). *Input* dari program harus bersifat .txt, namun *output* dapat bervariasi (.txt, .png. etc.).

Secara ringkas, algoritma yang dibawakan menggunakan konsep yang dimulai dengan eksplorasi seluruh kombinasi posisi dari seluruh N Queen. Setiap kombinasi yang dieksplorasi akan dicatat dan disimpan ke dalam array yang nantinya akan dilanjutkan ke proses validasi pada setiap kombinasi yang telah dicatat sebelum. Validasi akan melihat secara Horizontal, Vertical, Diagonal, dan Colors menyesuaikan persoalan. 

Program ini menyelesaikan 2 *bonus*, yaitu GUI dan *Output as an Image*. Untuk GUI, saya memanfaatkan *library* tkinter dan thread. Untuk *Output as an Image*, saya memanfaatkan PIL (*Python Imaging Library*) yang menggunakan konsep *screenshot* dan menyimpan foto tersebut sebagai *output*.

## Requirement & Instalasi Tertentu

Karena program ini memanfaatkan berbagai *library*, pengguna *program* kudu menjalankan *line of code* ini untuk menginstalasi beberapa hal.

Cukup mudah untuk menjalankan program ini. Dapat memencet tombol diatas kanan berbentuk Segitiga mengarah Kanan (*Run Code*). Namun, jika tidak bisa, kalian dapat menggunakan *terminal* dan menggunakan *line of code* sebagai berikut.
